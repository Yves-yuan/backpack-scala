# scala实现遗传算法解决0-1背包问题
## 测试数据
根目录下data文件夹下数据data1-data10。result.txt是每个测试数据的正确结果。
## 程序运行
GeneticBackpack.scala 是程序的入口,在代码中指定要读取的输入文件（data1-data10），然后直接运行程序。
程序默认设置运行时间为4秒钟，但是在测试数据中，要计算出正确答案一般只需要1~2秒。
欢迎补充更多的测试数据，欢迎hack!
